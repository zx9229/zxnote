### Linux安装Python35
先用epel-release，找不到python35的话，再用ius。
用ius的步骤
```
wget --output-document=auto_install_ius.sh  https://setup.ius.io/
bash auto_install_ius.sh
yum list | grep python35
yum list | grep python35 | grep setuptools
```
