#### 一些快捷键
|     键位     | 说明                                       |
| :----------: | :----------------------------------------- |
| Alt  + Shift | 切换`输入语言`                             |
| Ctrl + Shift | 切换`输入法`(同一`输入语言`之间的`输入法`) |
| Win  + Space | 切换`输入法`(不论`输入语言`)               |
